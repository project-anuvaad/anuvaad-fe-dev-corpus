File Content

Our Service allows you to upload PDF that contains english Judgments and Orders from Supreme Court of India and help you translate into 9 Indian vernacular languages.

Textual Content

Our Service allows you to input textual sentences or paragraphs primarily in English or Indian languages and get it translated to Indian vernacular languages and English respectively.

Application Features

Following section explains the features that is currently present in the application and is open for the registered user to use.

Translate

Using this function you can translate a single sentence from English to Vernacular language or Vernacular language to English. You can also compare the Anuvaad output against Google Translate output.

Interactive Translate

This feature enables you modify the received initial translated sentence to get translation sentences of user’s liking.

PDF Interactive Translate

The functionality enables user to upload a PDF document, select uploaded document language and desired output language needed from the drop down menu. The translation will be sometime to kindly upload the document and refresh the page to get to know the updates.
